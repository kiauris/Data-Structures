# Data Structures
Реализованы собственные структуры данных: 
- Стек
- Очередь 
- Списки
  - односвязный
  - двусвязный
  - циклический
- Множество
# Функционал

- Реализовано интерактивное консольное меню для работы со структурами

- Добавлен стресс-тест для проверки корректности и производительности

- Проведен анализ временной сложности операций и составлена таблица сравнения с теоретической сложностью
